# GR544P
A TAIHEN plugin for enabling a feature complete 544p resolution upgrade for Gravity Rush on the PS VITA

This plugin is region agnostic and will work for any Gravity Rush game region

This plugin functions like any TAIHEN plugin in terms of installation

To install this plugin, copy the GR544P.suprx file to ur0:tai/ folder on your modded vita, 

then enter your config file and add the following lines corresponding to your region so that the plugin will launch with the game

e.g.

*PCSA00011

ur0:tai/GR544P.suprx

*PCSF00024

ur0:tai/GR544P.suprx

*PCSD00035

ur0:tai/GR544P.suprx

*PCSD00003

ur0:tai/GR544P.suprx

THEN RESTART YOUR VITA
